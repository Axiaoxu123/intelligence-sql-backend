# sql-backend
> 智能sql生成管理系统

**为提高开发者工作效率而生！！！**

在线体验：[http://sql.xiaoxu9.cn](https://sql.xiaoxu9.cn)

## 技术栈
### 后端技术栈
- Spring Boot 2.7.x
- MyBatis Plus 3.5.x
- MySQL 8.x
- Spring AOP

### 前端技术栈
- React 18
- Umi 4.x
- Ant Design 4.x 组件库
- Ant Design Pro Components 高级组件
- TypeScript 类型控制
- Eslint 代码规范控制
- Prettier 美化代码
